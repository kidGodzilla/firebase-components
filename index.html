Components Library

Components Library

This is a collection of Components library examples utilizing bootstrap & firebase. It aims to provide a generic & simple API for creating dead-simple HTML components that are easy enough for beginners to understand yet complex enough to solve a wide-variety of common UI problems.

See a demo

HTML Includes

A declarative syntax to include HTML within any document.

Logic-less templates

Render a model into a logic-less handlebars template.

Live Binding

Elements are re-rendered automatically when the model changes.

Conditionals

Show or hide an element based on data inside a model.

Sorting, Limits & Offsets

Sorting & pagination are built-in to each component.

Repeating Components

Components can grow or shrink to match the number of items in a model.
